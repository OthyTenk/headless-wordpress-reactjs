## Getting Started

Cloning the repository:

```bash
git clone https://github.com/OthyTenk/headless-wordpress-reactjs.git
```

## Install packages

```bash
npm i
# or
yarn
# or
pnpm i
```

## Start the app

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
